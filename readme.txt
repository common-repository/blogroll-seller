A few notes about the sections above:

This plugin is for link sellers and allows them to add an expiry date for the blogroll links when either adding new or editing the anchors.
Basicaly it simply adds a field called expiry date (format mm/dd/yyyy) when you are adding/editing links. 
It automatically filters and removes expired links.

== Installation ==

1. Upload `blogroll-seller` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
